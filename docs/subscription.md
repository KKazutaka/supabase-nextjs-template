# サブスクリプション設計

## ステータス定義

| ステータス        | 説明                               |
| ----------------- | ---------------------------------- |
| `free`            | 無料プラン                         |
| `trial`           | トライアル中（Pro プラン）         |
| `active`          | 有料プラン課金中                   |
| `to_be_cancelled` | キャンセル予定（期限までは利用可） |
| `past_due`        | 課金失敗中                         |
| `incomplete`      | 初回決済未完了状態                 |
